# Soluciones de las guías de Pensamiento Computacional

Este repositorio contiene **soluciones de referencia** para las guías de ejercicios de la materia **Pensamiento computacional (CBC/UBAXXI)** de la UBA. Cada archivo corresponde a una unidad de la materia.

La idea de este repo es darte una mano con los ejercicios en los que te trabaste. Es imporante resaltar _soluciones de referencia_, **si con otra implementación llegaste al mismo resultado esta perfecto**.
### Estructura de los archivos:

En cada archivos te vas a encontrar con este formato por ejercicio:

\# Enunciado

... (Texto comentado)

\# Implementación

... (Código comentado)

Y también te podes encontrar con estos bloques, pero no están siempre:

\# Aclaración

... (Explicaciones adicionales)

\# Caso de uso

... (Llamado a la función)

### Recomendaciones
- Para correr algún ejercicio en particular, descomenta su implementación y su caso de uso (si tiene).
- Los ejercicios no están numerados, busca con Ctrl+F (CMD+F en mac) y pegá el enunciado del ejercicio.
- No descomentes todo al mismo tiempo porque puede causar confusión.
- Las aclaraciones y los enunciados dejalos siempre comentados.

### Unidades:

En la carpeta `primer_parcial`:

- Introducción a la algoritmia `intro_algoritmia.py`.
- Sentencias básicas y datos simples `sentencias_basicas.py`.
- Estructuras de control `estruc_control.py`.
- Secuencias, tuplas y listas `secuencias.py`.
- Recursos con listas `recursos_listas.py`.

En la carpeta `segundo_parcial`:
- Diccionarios `diccionarios.py`.
- Archivos `archivos.py`.
- Matplotlib `matplotlib.py`.
- Pandas `pandas.py`.
